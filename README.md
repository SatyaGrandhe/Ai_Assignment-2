Artificial Intelligence Lab Programs
Introduction

This project includes a collection of Artificial Intelligence programs developed using Python.
Each program illustrates important AI principles such as search techniques, verification mechanisms, and basic simulations of intelligent systems.

The objective of these programs is to provide practical knowledge of how AI methods are applied to solve problems and make logical decisions.

List of Programs
1. Turing Test Simulation

This program demonstrates a basic version of the Turing Test. In this simulation, a judge asks questions to both a human participant and a computer program.
The computer generates predefined responses that resemble machine-like behavior, while the human provides natural answers.
Based on the responses, the judge attempts to determine whether the replies came from a human or a machine.

Key Concepts:

Simulation of intelligent behavior

Pattern recognition

Decision analysis

2. CAPTCHA Authentication System

This program creates a random CAPTCHA consisting of letters and numbers.
The user must correctly enter the displayed CAPTCHA in order to confirm that they are a human user.

If the input matches the generated CAPTCHA, the system grants access; otherwise, it denies verification.

Key Concepts:

Random string generation

Human verification techniques

Simple security implementation

3. Breadth First Search (BFS)

This program implements the Breadth First Search algorithm for traversing graphs.
BFS explores nodes in layers, beginning from the starting node and moving outward level by level until the target node is located.

Key Concepts:

Graph searching

Queue data structure

Systematic node exploration

4. Depth First Search (DFS)

This program demonstrates the Depth First Search algorithm using a stack structure.
DFS follows one path deeply into the graph before backtracking and exploring alternative paths.

Key Concepts:

Graph traversal methods

Stack-based implementation

Backtracking strategy

5. Depth Limited Search (DLS)

Depth Limited Search is a modified version of DFS where the search is restricted to a specific depth.
If the goal node is deeper than the specified limit, the search process stops without continuing further.

Key Concepts:

Controlled depth searching

Recursive algorithms

Graph exploration

6. Iterative Deepening Depth First Search (IDDFS)

This algorithm merges the advantages of BFS and DFS.
IDDFS repeatedly applies Depth Limited Search while gradually increasing the depth limit until the target node is found.

Key Concepts:

Iterative searching approach

Depth expansion strategy

Memory-efficient searching

7. Water Jug Problem using BFS

This program solves the well-known Water Jug Problem by applying the Breadth First Search technique.
Two containers with fixed capacities are used, and the algorithm determines a sequence of operations to obtain the desired quantity of water.

Possible Operations:

Filling a jug

Emptying a jug

Transferring water between jugs

Key Concepts:

State space representation

BFS problem solving

Logical search techniques
